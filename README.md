# Student Result Management System

A simple Java mini project that uses JDBC to connect with MySQL and store student results.

# Features
- Enter student marks
- Calculate total, percentage, and grade
- Store data in MySQL database
- View student records

# Technologies Used
- Java
- JDBC
- MySQL
- VS Code

# Database Setup
- Create a MySQL database named `Mini`
- Create table `student_table`
- Update database username and password in code before running

# How to Run
- Add MySQL Connector/J to classpath
- Compile and run the program using terminal
